О проекте

Этот проект представляет собой инструмент для разработки своих генераторов команд, он имеет возможности:

  • Интеграция генератора в Python программы
  • Быстрое создание датапаков
  • Возможность добавлять интерфейс для более удобной генерации

Как использовать

этот проект подрузомивает возможности для разработки своих генераторов на основе библиотеки .
