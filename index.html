О проекте

MCCLIB - это мощный инструмент для разработки генераторов команд в Minecraft. Наш проект предоставляет следующие возможности:

  • Интеграция генератора в Python программы
  • Быстрое создание датапаков
  • Возможность добавлять пользовательский интерфейс для более удобной генерации

Как использовать

MCCLIB предоставляет широкие возможности для разработки собственных генераторов на основе нашей библиотеки. Вот простой пример использования:


from minecraft_commands import MinecraftCommand

mc_cmd = MinecraftCommand()
mc_cmd.set_block(10, 64, 10, "minecraft:diamond_block")
mc_cmd.teleport("Steve", 100, 64, 100)
mc_cmd.give_item("Alex", "minecraft:diamond_sword", 1)
mc_cmd.summon_entity("minecraft:zombie", 105, 64, 105)
mc_cmd.effect("Steve", "speed", 30, 2)
mc_cmd.clear_effect("Steve")

print(mc_cmd.execute())
